# To-Do List App

A simple to-do list web application that allows users to add, update, and delete tasks.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Persistent storage using `localStorage`
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
